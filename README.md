# 《我有一个朋友》

我记录了从小学为始，到大学、及大学毕业后几年为结束，二十年的人和事，共十二正篇、一后记，合计约10万4千字（含标点）。每篇文章基本都是兴致来了、一气呵成，难免有重复和纰漏，但我也再无精力做过多的修饰，写成什么样，就让它呈现什么样吧。
